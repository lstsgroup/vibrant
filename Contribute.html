Contribute¶

Contributions to vibrant are highly appreciated! If you consider contributing check out CONTRIBUTING.md.

Code style guideline¶

Please use fprettify with the following configuration if you want to submit code:

indent = 4
strict-indent = True
case = [2, 2, 2, 2]
whitespace = 2
whitespace-relational = False
line-length = 300
